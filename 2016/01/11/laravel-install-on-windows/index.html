部署

开发环境开发完成后，一般都会将软件部署到运行环境，部署类似应用，一般有三步：

  1. 将数据库导出，然后在目标系统里导入，修改数据库连接。
  2. 将目录整个打包，上传到目标系统。
  3. 微调。
    对于Laravel就是不匹配的配置需要修改下，我只修改了.env里面的数据库密码和config/app.php文件下的URL。
