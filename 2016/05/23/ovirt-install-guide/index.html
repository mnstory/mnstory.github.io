发表于 2016-05-23 |

添加主机

由于只有一台主机，主机又是管理机器又是实际做任务的机器，所以先把本机加入集群，进入管理页面后，左树依次展开找到“主机”，然后“新建”，填入本机参数（我的防火墙禁止了的，所以就不自动配置防火墙了）。

我的安装完后，失败了，状态一列显示：Install Failed，在点击“激活”变成了No Operational，点击“网络接口”/“设立主机网络”,为其分配网络后，状态变为UP。
