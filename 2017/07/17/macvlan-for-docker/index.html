发表于 2017-07-17 |

MACVLAN模式

MACVLAN有四种模式，分别是Bridge（默认），Private，VEPA(Virtual Ethernet Port Aggregator)，Passthru。
这些类型的区别，网上比较多，不细说，我只配置默认的Bridge模式。
